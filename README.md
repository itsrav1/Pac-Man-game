## 📝 Project Overview
This is a modern reconstruction of the classic **Pac-Man** arcade game. The project focuses on recreating the nostalgic gameplay experience with smooth mechanics and intelligent ghost AI.

### 🎮 How to Play
1. **Move:** Use the arrow keys (UP, DOWN, LEFT, RIGHT).
2. **Goal:** Eat all the yellow dots while avoiding the ghosts.
3. **Power-Ups:** Eat the large "Power Pellets" to turn the ghosts blue and gain the ability to eat them!

---

## 🚀 Key Features
* **Intelligent Ghost AI:** Each ghost has a unique personality/pathfinding logic.
* **Dynamic Scoring:** Track points for dots and ghost combos.
* **Original Sound Effects:** Integrated classic sounds for an immersive experience.
* **Collision Detection:** Precise hit-box management for walls and entities.

## 🛠️ Technical Implementation
* **Logic:** Implemented a grid-based movement system to ensure Pac-Man stays within the maze boundaries.
* **State Management:** Handles "Ready", "Playing", "Dying", and "Game Over" states.
